# AI Resume Analyzer 🚀

## Description
A web application that analyzes resumes by extracting text, identifying key skills, and providing suggestions for improvement.

## Features
- Upload resume (PDF)
- Extract text using Python
- Detect skills
- Provide suggestions
- Clean web interface

## Tech Stack
- Python
- Flask
- HTML & CSS

## How to Run
1. Install dependencies:
   pip install flask PyPDF2

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
